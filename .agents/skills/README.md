# .agents/skills compatibility layer

Эта директория хранит официальный слой совместимости для стандартного пути skills.

- Канонические навыки расположены в `.trae/skills`.
- Таблица соответствия находится в `.agents/skills/skills_map.json`.
- При изменении набора skills обновляйте карту соответствия и `docs/SKILLS_INDEX.md`.
